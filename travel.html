WELCOME TO MY PORTFOLIO

Gadget Junkie

Gadget Junkie Is A Simple E-Commerce Website Built With React. It Is A Responsive Website That Can Be Viewed On Any Device. It Has A Cart Functionality That Allows Users To Add And Remove Items From The Cart.

Jobster

This Is A Simple Job Application Tracker Built With React And NodeJs Backend That Allows Users To Keep Track Of Their Job Applications. Users Can Add, Edit, And Delete Job Applications.

Nasa Mission Control

A Web Application That Allows Users To Schedule A Mission To One Of The Kepler Exoplanets. The Application Is Built Using The MERN Stack And Arwes- A Futuristic Sci-Fi UI Web Framework.

Oregon Hikes

A Web Application That Allows Users To Add Their Favorite Hiking Trails In Oregon. The Application Is Built Using NodeJS. Users Can Add, Edit, And Delete Hiking Trails. Authors: Richard Brown

Node Fitness

A Web Site For A Fitness Center Built Using NodeJS, EJS, MONGODB, and Express. Users sign up and send emails and read about fitness and good eating habits. Authors: Richard Brown

Locations

This Website Was Done While I Was In My Coding Bootcamp. This App Was Designed To List Travel Locations Around The World. Users Login, Sign Up, Create New Locations, Like Locations,Heart Locations, And Sort Locations. Authors: Thor Kuivanen, Richard Brown, And Troy Hancock.

Blog

This Is A Blog Website Built With EJS And A NodeJs, MONGODB Backend That Allows Users To Create, Edit, Sign Up, And Delete Blog Posts. The website is responsive and can be viewed on any device. Authors: Richard Brown

Music App

This Is A Music App Built With React And A NodeJs, MONGODB Backend That Allows Users To Create, Edit, And Delete Song Playlists. Authors: Richard Brown And Gianni Nola

Terminal Chat

This Is A Terminal Chat App Built With NodeJs And Socket.Io. It Allows Users To Chat With Each Other In Real Time.

Bitcoin And Crypto News

This App Was Designed To List Bitcoin News.Users Can Comment, Check Crypto Prices, Link To Full Article,And Read About Bitcoin History.

The Plants

This App Was Built With React And Allows Users To View Information About The Planets In Our Solar System.
